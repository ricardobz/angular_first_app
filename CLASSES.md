# Module 1

## Selectors


```javascript
selector: 'app-servers'   // -> <app-servers></app-servers>
selector: '[app-servers]' // -> <div app-servers></div>
selector: '.app-servers'  // -> <div class="app-servers"></div>
```


## String interpolation
```html
<p>{{ title }}</p>
```

## Property binding

```html
<p [innerText]="title"></p>
```
